🚀 HOW TO START
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install dependencies:
   pip install -r requirements.txt

2. Configure .env file:
   - Supabase credentials
   - Gemini API key
   - Email credentials (Gmail)
   - Razorpay keys (optional)

3. Run database schema:
   Execute supabase-advanced-schema.sql in Supabase

4. Start Flask backend:
   python app_advanced.py
   
   You'll see:
   🚀 MedAI Advanced Backend Starting...
   🤖 AI Inventory Agent started
   🌐 Server running on http://localhost:5000

5. Open frontend:
   - Open index.html with Live Server
   - Or: python -m http.server 8000

6. Test the system:
   - User: Login → Shop → Checkout → Order
   - Admin: Login → Manage Inventory → View Orders
   - AI: Wait for low stock detection!
